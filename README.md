# AI-Travel-Planner
An AI-powered Travel Planner built with React and Flask. It uses a Kaggle dataset of Indian tourist attractions to rank and recommend places using interest matching, rating-based scoring, and budget constraints. The app generates a day-wise itinerary, shows estimated cost breakup, and provides Google Maps navigation links.
# AI Travel Planner (Kaggle Dataset)

React + Flask web app that generates a day-wise itinerary using a Kaggle dataset of Indian tourist places.
It recommends places using interest + rating ranking and budget optimization, and provides cost breakup and map links.

## Run Backend
cd backend
venv\Scripts\activate
python app.py

## Run Frontend
cd frontend/travel-ai
npm install
npm run dev
